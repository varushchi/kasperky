Тестовое задание сделано из двух частей:
1. сервер: на сервере расположены файл JSON, реализация поиска по всем полям и сортировка определнного поля по возростанию или убыванию;
2. клиент: на клиенте выполнено оформление.
Чтобы начать работу нужно открыть 2 терминала: 
1. 
    cd server 
    npm install 
    node index.js
2. 
    cd client
    npm install
    npm start
